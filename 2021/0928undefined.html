算法动态规划思想总结

数据结构与算法
学习笔记
发布日期:   2021-09-28

一、动态规划三大步骤

第一步骤：定义数组元素的含义，我们会用一个数组，来保存历史数组，假设用一维数组 dp[] 吧。这个时候有一个非常非常重要的点，就是规定你这个数组元素的含义，例如你的 dp[i] 是代表什么意思？

第二步骤：找出数组元素之间的关系式，当我们要计算 dp[n] 时，是可以利用 dp[n-1]，dp[n-2]…..dp[1]，来推出 dp[n] 的，也就是可以利用历史数据来推出新的元素值，所以我们要找出数组元素之间的关系式。

第三步骤：找出初始值。有了初始值，并且有了数组元素之间的关系式，那么我们就可以得到 dp[n] 的值了，而 dp[n] 的含义是由你来定义的，你想求什么，就定义它是什么，这样，这道题也就解出来了。

二、常见题目

1.跳台阶

一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法。

1、定义 dp[i] 的含义为：跳上一个 i 级的台阶总共有 dp[i] 种跳法

2、一种是从第 n-1 级跳上来，一种是从第 n-2 级跳上来，所以有 dp[n] = dp[n-1] + dp[n-2]

3、初始条件，数组不允许下标为负的值，因此直接给出 dp[0] = 0,dp[1] = 1,dp[2] = 2

int f( int n )&#123;
    if(n <=1) return n;
    //创建数组
    int dp[] = new int[n+1];
    //给出初始值
    dp[0] = 0；
    dp[1] = 1；
    dp[2] = 2；
    //通过关系式进行计算
    for(int i;i < n;i++)&#123;
        dp[i]=dp[i-1]+dp[i-2];
        &#125;
    return dp[n];
&#125;

2.求路径

一个机器人位于一个 m x n 网格的左上角 ，机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角，问总共有多少条不同的路径？

1、当机器人从左上角走到(i, j) 这个位置时，一共有 dp[i][j] 种路径。因此题目所求为dp[i-1][j-1]

2、一种是从 (i-1, j) 这个位置走一步到达，一种是从(i, j - 1) 这个位置走一步到达，把所有可能走的路径都加起来，所以关系式是 dp[i][j] = dp[i-1][j] + dp[i][j-1]

3、i 和 j 不能为负数，那么可以用 i 和 j 等于0作为初始值。

dp[0][0….n-1] = 1；//相当于最上面一行，机器人只能一直往右走

dp[0…m-1][0] = 1; // 相当于最左面一列，机器人只能一直往下走

public static int Solution(int m,int n)&#123;
    if(m <= 0 || n <= 0)&#123;
        return 0;
    &#125;
    //初始值
    int dp[][] = new int[m][n];
    for(int i;i < m;i++)&#123;
        dp[i][0] = 1;
    &#125;
    for(int j;j < n;j++)&#123;
        dp[0][j] = 1;
    &#125;
    //关系式计算
    for(int i = 1;i < m;i++)&#123;
        for(int j = 1;j < n;j++)&#123;
            dp[i][j] = dp[i-1][j]+dp[i][j-1]; 
        &#125;
    &#125;
    return dp[m-1][n-1];
&#125;

3.矩阵最小路径和

给定一个包含非负整数的 m x n 网格，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。每次只能向下或者向右移动一步。

1、走到( i , j )这个位置时，最小路径和为dp[i][j],因此题目所求为

dp[i-1][j-1]

2、计算从(i-1,j)和(i,j-1)哪个路线是最小的，因此关系式

dp[i][j] = min(dp[i-1][j],dp[i][j-1])+arr[i][j]

3、同样，i 和 j 都不能为0，我们用最上面一行与最左边一列作为初始值

dp[0][j] = arr[0][j] + dp[0][j-1]; // 相当于最上面一行，机器人只能一直往右走

dp[i][0] = arr[i][0] + dp[i-1][0]; // 相当于最左面一列，机器人只能一直往下走

public static int Solution(int[][] arr)&#123;
    int m = arr.length;
    int n = arr[0].length;
    if(m <= 0||n <= 0) return 0;
    int dp[][] = new int[m][n];
    //初始值
    dp[0][0] = arr[0][0];
    for(int i = 1;i < m;i++)&#123;
        dp[i][0] = arr[i][0] + dp[i-1][0];
    &#125;
    for(int j = 1;j < n;j++)&#123;
        dp[0][j] = arr[0][j] + dp[0][j-1]; 
    &#125;
    //关系式计算
    for(int i = 1;i < m;i++)&#123;
        for(int j = 1;j < n;j++ )&#123;
            dp[i][j] = Math.min(dp[i-1][j],dp[i][j-1]) + arr[i][j];
        &#125;
    &#125;
    return dp[m-1][n-1];
&#125;

4.连续子数组最大和

给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

1、对于这道题， dp[i] 的含义是以 nums[i] 结尾的子序的最大和。那么我们要找的答案就是

max = max(dp[0], dp[1],….., dp[n-1])

2、当dp[i-1]是正数，那就要，当dp[i-1]是负数，那就不需要。因此关系式为

dp[i]=max(dp[i-1]+nums[i],nums[i])

3、当只有一个元素时，初始值为

dp[0]=nums[0]

public class Solution&#123;
    public int Sub(int[] nums)&#123;
        int len = nums.length;
        if(len == 0) return 0;
        int[] dp = new int[len];
        dp[0] = nums[0];
        int max = dp[0];
        for(int i = 1;i<len;i++)&#123;
            dp[i] = (dp[i-1] + nums[i] > nums[i]) ? dp[i-1]+nums[i] : nums[i];
            if(dp[i] > max)&#123;
                max = dp[i];
            &#125;
        &#125;
        return max;
    &#125;
